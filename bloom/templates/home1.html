Welcome to the Outcome Mapping Tool!

About the Project
Our Outcome Mapping Tool is designed to help educators and academic institutions automate the process of mapping questions to Course Outcomes (COs), Program Outcomes (POs), and Bloom’s Taxonomy Levels. This tool leverages Natural Language Processing (NLP), using advanced machine learning techniques to analyze question content and assign educational outcomes based on similarity and cognitive complexity.

Project Goals

  1. Automate Outcome Mapping: Quickly and accurately assign COs, POs, and Bloom’s Levels to questions, saving educators significant time and ensuring objective alignment.
  2. Enhance Assessment Quality: By consistently linking questions with learning objectives, educators can create assessments that better reflect course and program goals.
  3. Support Educational Transparency: The system clearly displays mapped outcomes for each question, making it easier to understand and track how assessments align with learning expectations.

Key Features

  • Automated CO and PO Mapping: Using state-of-the-art NLP models, the tool automatically identifies the most relevant Course Outcome and Program Outcome for each question.
  • Bloom’s Taxonomy Classification: The tool assigns a Bloom’s Level (Remember, Understand, Apply, etc.) based on the question's cognitive complexity, aligning it with the desired educational objective.
  • Customizable for Multiple Courses: Supports mapping across various subjects, enabling institutions to use it across departments.

How It Works

  1. Question Input: Users can input questions directly into the system.
  2. NLP-Powered Analysis: The tool compares the question’s intent and content with CO descriptions using Sentence-BERT embeddings.
  3. Outcome Display: Within seconds, the system presents the mapped CO, PO, and Bloom’s Level for the question, helping educators immediately assess its alignment with learning objectives.

Who Can Benefit?
This tool is perfect for:

  • Educators and Faculty: Simplify the process of aligning assessments with educational goals.
  • Institutions: Provide a unified approach to tracking and verifying learning outcomes.
  • Accreditation Bodies: Enhance documentation for accreditation by showing evidence of assessment alignment.

Explore the tool to experience how advanced AI-driven solutions can streamline educational assessments, improve alignment with learning goals, and create a more transparent, efficient approach to outcome mapping!
